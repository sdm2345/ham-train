> 电源功率计算：**P_supply = U × I = (N / η) 反推** 

**公式推导**：
- 输出功率 P_out = N (瓦)
- 电源效率 η = 80% = 0.8
- 电源功率 P_supply = N / 0.8 = 1.25N (瓦)
- 工作电流 I = P_supply / U = 1.25N / 13.8 = **0.091N (安)** ✓

| 已知条件 | 单位 |
|---------|------|
| 工作电压 U | 13.8V |
| 射频功率 P_out | N瓦 |
| 电源效率 η | 80% |
| 求：I | 0.091N安 |
