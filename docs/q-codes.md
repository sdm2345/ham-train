## 四、记忆口诀

**QR 开头 → 操作/状态类**
- `QRT` = **R**est / **T**erminate → 关机再见
- `QRU` = **U**nfinished → 无（nothing）
- `QRV` = **V**oilà / **V**oid → 准备好了
- `QRM` = **M**an-made → 人为干扰
- `QRN` = **N**ature → 天电干扰
- `QRL` = **L**ocked → 正在忙（频率占用）
- `QRP` = 小功率（P = Power↓）
- `QRO` = 大功率（O > P，功率更大）
- `QRQ` = 快（Q = Quick）
- `QRS` = 慢（S = Slow）
- `QRZ` = 呼叫我的是谁（Z = 最后字母，"终于谁在叫？"）
- `QRH` = 频率漂移（H = Hop）
- `QSB` = 信号衰落（B = 起伏波浪）
- `QSD` = 键控缺陷（D = Defect）

**QS 开头 → 联络/通信类**
- `QSL` = 确认（L = Log 记录）
- `QSO` = 通联（O = Open communication）
- `QSP` = 转信中继（P = Pass / Proxy）
- `QSY` = 换频（Y = 转向另一个频率）
- `QSK` = 全双工 CW 插入（K = Key）

**QT 开头 → 位置/时间类**
- `QTH` = 位置（H = Here）
- `QTR` = 时间（R = Real time）

## 五、易混对比

| 对比组 | 区别 |
|--------|------|
| **QRM vs QRN** | QRM = 人为干扰（Man-made）；QRN = 天电干扰（Nature） |
| **QRP vs QRO** | QRP = 降功率；QRO = 升功率 |
| **QRS vs QRQ** | QRS = 减速（Slow）；QRQ = 加速（Quick） |
| **QSO vs QSP** | QSO = 直接通联；QSP = 帮助中转 |
| **QSL vs QSO** | QSL = 确认抄收；QSO = 表示通联本身 |
| **QRL vs QRU** | QRL = 我正忙（频率占用）；QRU = 我无事（结束联络） |
| **QTH vs QRA** | QTH = 物理位置；QRA = 电台名称/呼号 |
| **QRV vs QRT** | QRV = 准备好开始；QRT = 关机结束 |
